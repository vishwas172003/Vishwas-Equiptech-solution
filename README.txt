Vishwas Equiptech Solutions Website

How to view:
1. Right-click ZIP and choose Extract All.
2. Open the extracted folder.
3. Double-click index.html.

Important: Do not open index.html directly from inside the ZIP, because videos/photos may not load properly.

To deploy on Vercel:
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Import the repository in Vercel.
4. Add your domain in Vercel > Settings > Domains.
